In order to execute the first part of the program:

1. Make sure the files are extracted
2. Open up this directory in the command line
3. Check java is installed (type "java -version")
- If java is not installed, go here to install the jre: http://docs.oracle.com/javase/7/docs/webnotes/install/
4. In the command line, type "java Main"
- This will run the main program that contains directions
on how to run and test every part of this project
5. Enter a number for how many decimals the matrix answers should
be in
6. Enter 1 to enter part 1
7. Select from a list of part's programs to run and follow the directions

The written component is in the file "writtenpart1.docx"
